<static> handlePuzzleLoad(event)

Callback function that executes after all files in the manifest have loaded. This function will pop all the pieces that have been added to the stack and add them to the puzzle, wrapped in a PieceContainer. This function also sets the matches of all the pieces, as defined in the loadObject.
