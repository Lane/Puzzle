Class: Point

Point

new Point(piece, x, y)

Creates a point on a piece that can be used to connect two pieces together with a PointMatch
Parameters:
Name Type Description
piece Piece The piece the point is associated with
x int The horizontal offset from the centre of the piece
y int The vertical offset from the centre of the piece
Properties:
Name Type Description
x int Horizontal offset from the centre of the piece
y int Vertical offset from the centre of the piece
angle int The angle, in degrees, that is formed by the x axis and a line segment from the origin to the point.
piece Piece The piece the point is associated with
radius int Length of the line segment drawn from the origin to the point.
match Point The matching point that this point connects to
_stageOffset Object An object containing the x and y offset of the Point relative to the stage
